Detailed Description

Header file for the sequence_of_longs module, containing the ADT (Abstract Data Type) with the same name: sequence_of_longs.

This header file declares the basic structure used to store the sequences of longs and all the routines used to manipulate these sequences.

void SEQL_add ( struct sequence_of_longs *  sl,
long  new_term 
)

Adds a given value as a further term of the given sequence of longs.

Parameters
slA pointer to the sequence of longs to which the new term will be added.
new_termThe new term to add to the sequence.
Postcondition
The sequence contains the same terms it contained before, in the same order, plus the new term in its last position.
Precondition
sl ≠ null

The time required to add the new term depends on the non-observable state of the sequence. If this time is amortized over the series of additions prior to addition being performed, then this time is constant with regards to the length of the sequence.
