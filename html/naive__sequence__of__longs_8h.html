Detailed Description

Header file for the naive_sequence_of_longs module, containing the ADT (Abstract Data Type) with the same name: naive_sequence_of_longs.

This header file declares the basic structure used to store the sequences of longs and all the routines used to manipulate these sequences.

void NSEQL_add ( struct naive_sequence_of_longs *  sl,
long  new_term 
)

Adds a given value as a further term of the given sequence of longs.

Parameters
slA pointer to the sequence of longs to which the new term will be added.
new_termThe new term to add to the sequence.
Postcondition
The sequence contains the same terms it contained before, in the same order, plus the new term in its last position.
Precondition
sl ≠ null

The time required to add the new term varies linearly with the length of the sequence.
